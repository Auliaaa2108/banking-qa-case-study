# Banking QA Portfolio

Selamat datang di portofolio QA saya. Fokus utama saya adalah pada **Banking Domain Testing**, di mana integritas data dan keamanan adalah prioritas tertinggi.

## Tentang Saya
Saya adalah seorang Quality Assurance yang berfokus pada pengujian sistem perbankan. Saya memiliki keahlian dalam:
- **Test Planning & Strategy** (Khususnya fitur Transfer Dana)
- **API Testing** (Postman)
- **Database Validation** (SQL)
- **Bug Reporting & Lifecycle Management**

## Struktur Portfolio
- `/test-plan/`: Berisi dokumen skenario pengujian transfer dana.
- `/sql-queries/`: Contoh kueri SQL untuk validasi saldo nasabah.
- `/api-testing/`: Koleksi API untuk pengujian integrasi transaksi.
- `/bug-reports/`: Contoh pelaporan bug yang standar dan detail.
